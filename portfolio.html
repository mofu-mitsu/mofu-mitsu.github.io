Projects & Case Studies

Cocotte - Simple Memo App

制作期間: 約2ヶ月 個人開発
React Vite Supabase Vercel PWA
概要

デバイスごとに完全分離された、シンプルで可愛らしいPWAメモアプリです。色分け、フォルダ管理、画像添付などを実装。

工夫した点
  • SupabaseのRLS（Row Level Security）ポリシーを利用し、ユーザー登録なしでデバイスIDごとにデータを完全分離。
  • QRコードを利用した、セキュアで直感的なデバイス間データ同期（ログイン）機能の実装。
  • ファイルアップロード（画像添付）機能の対応。
苦労した点と解決方法
  • データ型の不一致によるビルドエラーが発生 → データベースのスキーマを整理・再定義することで解決。
  • RLSポリシーの複雑な設計 → 「デバイスID」を基準としたポリシーに再設計することで、堅牢性と利便性を両立。

地雷系黒猫みりんてゃ UTAU音声ライブラリ

音声収録・編集 Python (Bot開発)
UTAU Audio Processing Python
概要

公式キャラクター「みりんてゃ」の歌唱用音声ライブラリを独自制作。あわせて、Pythonを用いたSNS運用Botも開発。
