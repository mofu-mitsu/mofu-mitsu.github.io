Project Case Studies

GA4連携 自動ランキング生成システム

Python / GitHub Actions / Google Analytics API
課題

静的サイト(GitHub Pages)において、手動更新の手間を省きつつ、常に最新の人気コンテンツをユーザーに提示したい。

解決策

PythonスクリプトでGA4のAPIを叩き、PV数上位を取得。GitHub Actionsで毎日1回自動実行し、JSON形式でランキングを動的に出力。

Deep Cognition Archive

JavaScript / HTML5 / CSS3
技術的工夫

単なる選択肢の集計ではなく、回答時の「ミリ秒単位の迷い」や「修正行動」をログ化。情報代謝プロセスを定量的に分析するロジックを実装。

成果

心理学ガチ勢から高い評価を獲得。独自性の高いアルゴリズムにより、従来の簡易診断とは一線を画す精度の高さを実現。

Cocotte - Simple Memo App

付加機能
